CESM CASEROOT env_case.xml file

Model Version: CESM2.0
HTML created on: 2017-06-21

Description

The CASEROOT scripts read the XML settings in the env_case.xml file in order to
determine case specific settings.

Interface Tools

CESM2.0 supports multiple case groups which can be queried but not changed. The xmlquery script tool
accepts the --subgroup command line argument to define one of the case groups.
