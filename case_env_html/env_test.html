CESM CASEROOT env_test.xml file

Model Version: CESM2.0
HTML created on: 2017-06-21

Description

The CASEROOT scripts read the XML settings in the env_test.xml file in order to
determine case specific settings when running a regression test..

Interface Tools

CESM2.0 supports a single test group with id="test" in env_test.xml which can be queried but not changed. The xmlquery script tool
